IntelliJ IDEA 단축키

단축키 정리
Alt + Enter 무엇이든 수정 가능하고 오류, 경고, 제안 중복 유형 제거 누르면 더 나은 제안을 보여주고 수락가능
F2 오류, 경고, 제안으로 이동가능 그 다음 Alt + Enter 눌러주면 수정 가능
Cmd⌘ + 1 프로젝트 창을 열고 초점을 이동과 방향키를 사용해 트리 탐색하고 타이핑을 통해 검색가능
ESC 프로젝트 창에서 에디터로 다시 초점을 옮겨주거나 변경 사항을 적용하지 않고 모든 팝업을 닫아줌
Cmd⌘ + E 가장 최근 열었던 파일 확인
Cmd⌘ + B 코드 안에서 다른 코드로 이동할 뗴, 심볼의 선언으로 이동 가능 클래스 이름에서 cmd+B를 누르면 클래스 파일로 이동 Alt + Cmd + B 구현탐색도 가능
Alt + F7 사용위치 검색할 때 인터페이스 이름에서 사용하면 필드 선언 또는 인터페이스를 구현한 클래스를 비롯하여 해당 인터페이스가 사용된 모든 위치가 검색창에 표시
Ctrl * 2 IDE에서 어떤 위치에 있든 혹은 어떤 파일이 열려 있든, ctrl을 두 번 누르면 Run Anyting창이 열림
Alt + 방향키 커서 주변의 코드 섹션을 확장하거나 축소 가능
Cmd⌘ 주석추가 가능, 전체 블록을 주석처리 할 때는 Alt + Cmd + /
Shift + Cmd⌘ + Enter 현재 구문 완성 기능
Alt + Cmd⌘ + L 현재 파일의 서식을 프로젝트 표준에 맞춰주고, 파일에서 변경된 줄의 서식을 지정하는 데 사용될 수 있음
Ctrl + T 모든 리팩토링을 표시하는 것이 가능
Shift + Cmd⌘ + A IntelliJ IDEA의 모든 액션을 검색가능, 설정도 검색 가능
Shift * 2 무엇이든 검색 가능한 검색 창이 표시됨
